# EducateBuddy

web based collaboration platform for university students.

## How to run the website
1. download this repository
2. Open index.html in a browser
3. Or visit the live version: https://educatebuddy-d8460.web.app 

## Firebase
This project uses Firebase. The configuration is included in app.js.

## Features
- User authentication with email verification
- Group creation and joining via unique codes
- Real time chat
- File sharing
- Task management with deadlines and assignment
- Calendar view of all deadlines
- User profiles
- Notifications
